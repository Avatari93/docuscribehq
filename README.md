## About DocuScribe

Signing documents digitally should be fast, easy, and the preferred method for all signatures worldwide. While this technology is now readily accessible, it introduces a new element into every transaction: the signing tool providers. Though this involvement isn't inherently problematic, it raises important questions about how we want these providers to operate and maintain trust.

We invite you to help us build the next generation of open trust infrastructure‚Äîone that ensures transparency, security, and fairness in every digital signature.
